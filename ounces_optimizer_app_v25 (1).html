What does this app do?

The Ounces Optimizer helps you decide which stopes and development headings to haul ore from — and how many trucks to assign to each — to hit your daily ounces target.

How ounces are calculated

Ounces = Tonnes x Grade x Ounces Factor

Where:

  • Tonnes = Trucks x Truck Factor (51 t/truck)
  • Grade = g/t of the stope or development heading
  • Ounces Factor = 0.03215
  • Development: 1 cut = 350 t | Default grade = 5.7 g/t

How to use

  • Set Ounces Target, Plan Tonnes, Ounces Factor
  • Add ore stopes with Stock and Grade
  • Add development headings by entering number of cuts
  • Edit any stope or heading inline using the pencil button
  • Toggle ON/OFF and set Cap to limit trucks per source
  • Tap ☁ Publish to push changes to all devices
  • Tap Find Best 3

Stope Setup & Ring Ranges

The Stope Setup tab is where you record each stope's full monthly ring data once: ring name, grade, and tonnes for every ring in that stope. This becomes the master reference the Optimizer pulls from.

In the Optimizer's Add Stope form, select a stope from Setup, then enter either:

  • A single ring (e.g. R03) — pulls that ring's exact tonnes and grade
  • A ring range (e.g. R01-R04) — combines all rings in that range and computes a tonnage-weighted average grade
Average Grade = Σ(Ring Tonnes x Ring Grade) / Σ(Ring Tonnes)

This means richer, larger rings pull the blended grade more strongly than smaller or leaner ones — the same way actual mill feed grade behaves when stock from multiple rings is mixed.

Material Movement Constraints (Underground Mine)

Obuasi is an underground operation, so getting broken ore and waste to surface isn't just about truck count — it's also about where that material can physically go and how fast it can get there. The Destinations tab models this.

Ore Passes: KRS & KMS

Ore passes are vertical conduits between levels. Trucks tip material at an upper-level tipping point; it falls through a chute at a lower draw point onto a conveyor, which feeds a skip that hoists it to surface.

  • KRS has two tips — Tip 6 (1,000 t capacity) and Tip 7 (3,500 t capacity). Tipping points sit on 2400L; draw points and chutes are on 2600L.
  • Either tip can handle ore or waste, switched according to operational priority. Switching material types requires the tip to be fully emptied first.
  • KMS is currently unavailable due to an incident. It remains in the system so it can be re-enabled once it returns to service.

Surface Destinations

Material that doesn't go through an ore pass can be hauled directly to surface stockpiles:

  • Ore Pad — surface stockpile for ore
  • Waste Dump — surface dump for waste rock

Why a destination can be unavailable

  • Changing material — the tip is being emptied before switching between ore and waste
  • Maintenance — scheduled servicing of the chute, conveyor, or skip system
  • Breakdown — unplanned equipment failure
  • Full — the tip has reached its tonnage capacity and must be drawn down before it can accept more

Mark a destination unavailable in the Destinations tab and select the reason — this keeps everyone on the same page about where material can and can't be tipped right now.

The 18-Hour Working Day

Trucking doesn't run a full 24 hours — the operational day is 18 hours, split into two 9-hour shifts. All haulage capacity calculations (max trucks per route, per level, per destination) are based on this 18-hour window, not a full day.

Max Trucks/Day = 18 hrs ÷ Cycle Time (hrs)

Cycle time is the average time for a truck to load, haul, dump, and return for a given level-to-destination route. The Haulage Capacity Calculator in the Destinations tab lets you pick any level and destination to see the maximum trucks — and tonnes — that route can realistically support in a day.

If a level shows no cycle time for a destination, it means material movement hasn't started from that level to that destination yet, so no haulage rate exists for it.
